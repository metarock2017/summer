## 飞机大战游戏介绍

1.采用原生js开发

2.目前仅完成基本功能，4个难度模式

### 好的设计思路

1.由函数内部调用函数，代码清晰明了

2.整个js代码位于函数表达式中，无全局变量

3.大量使用闭包，子弹，敌机的运动都是闭包调用

4.事先把关卡难度，敌机生成时间，我军子弹发射速度存数组，直接使用，避免了switch判断

5.封装碰撞检测 、获取元素offset值的函数
